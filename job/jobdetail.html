Senior Frontend Developer

GlobalDesign Inc.

San Francisco, CA (Remote) $120,000 - $150,000 Posted 2 days ago
Full-time Remote Senior Level

Job Summary

GlobalDesign Inc. is seeking a Senior Frontend Developer to join our dynamic team. In this role, you'll work closely with our designers and backend developers to create beautiful, intuitive web experiences for our global client base. You'll have the opportunity to lead projects, mentor junior developers, and contribute to our open-source initiatives.

Key Responsibilities

  • Develop and maintain responsive, cross-browser compatible web applications using modern frontend technologies
  • Collaborate with UX/UI designers to transform wireframes and designs into functional user interfaces
  • Optimize applications for maximum speed and scalability
  • Implement state management solutions and ensure data consistency across the application
  • Write clean, maintainable, and well-documented code
  • Conduct code reviews and provide constructive feedback to team members
  • Stay updated with industry trends and emerging frontend technologies
  • Mentor junior developers and contribute to team knowledge sharing
  • Work in an agile environment with regular sprints and deliverables

Requirements

  • 5+ years of professional experience in frontend development
  • Expert knowledge of JavaScript, HTML5, and CSS3
  • Strong experience with React.js and its ecosystem
  • Proficiency with modern CSS frameworks, preferably Tailwind CSS
  • Experience with state management libraries (Redux, Context API, etc.)
  • Understanding of cross-browser compatibility issues and solutions
  • Knowledge of RESTful APIs and GraphQL integration
  • Familiarity with version control systems (Git)
  • Experience with testing frameworks (Jest, React Testing Library)
  • Strong problem-solving skills and attention to detail
  • Excellent communication and teamwork abilities

Education & Experience

  • Bachelor's degree in Computer Science, Web Development, or related field (or equivalent practical experience)
  • 5+ years of professional experience in frontend development
  • Previous experience leading development projects is a plus
  • Portfolio of web applications or websites demonstrating your skills

Required Skills

React JavaScript HTML5 CSS3 Tailwind CSS Redux TypeScript Git Jest RESTful APIs
